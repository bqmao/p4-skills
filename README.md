# P4 Skills

A collection of AI agent skills for working with Perforce (P4) version control.

[繁體中文](README.zh-TW.md)

## Skills

| Skill | Description |
|---|---|
| [p4-workspace-check](skills/p4-workspace-check/SKILL.md) | Validates and auto-corrects the active P4 client workspace before running commands |
| [p4-duplicate-stream](skills/p4-duplicate-stream/SKILL.md) | Clones a P4 stream hierarchy under new names |
| [p4-export](skills/p4-export/SKILL.md) | Exports all files from a changelist to a local directory |
| [p4-move-conflict-files](skills/p4-move-conflict-files/SKILL.md) | Isolates unresolved conflict files into a new pending CL |
| [p4-port-cl](skills/p4-port-cl/SKILL.md) | Ports a submitted CL's changes to other workspaces/streams |
| [p4-claude-simplify](skills/p4-claude-simplify/SKILL.md) | Multi-agent code review and cleanup of open P4 files |
